Forms

Forms allow users to send data. It uses the "form" tags. It has an opening and closing tag.
An HTML form sends the input variables to an HTTP request, containing the data entered to a server.
Different types of inputs are:

  • radio buttons
  • number
  • checkbox
  • textarea
  • text
  • password
  • email
A form can also have action attributes that points the form where to go. When there is no path, it just sends the form right back here.
ex.) form action = " / registration " method = "POST">
This will go to the regristration path.


Within that form tag, you can also see a method.
  • GET: retrieves information from the server
  • POST: sends information to the server
