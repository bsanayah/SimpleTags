What are Anchors and why are there pictures of anchors when I click this?

This is an anchor! Anchors link different sites together.

  • "a" : is the tag for the anchor element. It has an openeing and closing tag.
  • "href" : is what file we are anchoring. href is an attribute that means hypertext reference
    • -Then you put the filename inside the quotation marks "...".
  • If you want to label the anchor, you can! Put the name inside the anchor tag.




These are pictures of an anchor.
Images are displayed with the images element (img tag)

  • "src" : is an attribute that will find the source of the image (my source is a filename within my project folder.)
  • "alt" : is a an attribute that makes the alternative text element.
    • -Describe your image within this tag so screen readers can read it. More accessible!
