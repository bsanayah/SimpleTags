Tables!

Tables are used to display an arragement of data.

  • "table" : a tag that tells us we will start a table.
    • -It has a closing and end tag.
  • "tr" : table row is a tag that outlines the rows of a table.
    • -nested inside the table tag. -It has a closing and end tag.
  • "th" : table heading is a tag that titles the rows of tables
    • -nested inside the table rows (tr)
  • "td" : table data is a tag that houses the data.
    • -nested inside the table rows (tr)
    -It has a closing and end tag.
